一些学习过程中遇到的常用命令

一些学习过程中遇到的常用命令，比如Linux、hexo、git等

一些常用命令

使用wget下载clangd

ubuntu2004的软件源安装clangd版本太低，用wget安装，随便还可以使用clang编译器了

nfs文件系统命令

服务端：

查看共享文件夹

更新nfs

sudo exportfs -arv

修改配置文件

sudo gedit /etc/exports

客户端：

挂载：sudo mount -t nfs 192.168.1.101:/home/sakura/workdir /mnt

查看：showmount -e 192.168.43.27

mount -t nfs -o nolock,vers=3,port=2049,mountport=9999 192.168.1.105:/home/sakura/worldir /mnt

hexo博客命令

新建操作
新建文章 hexo n “文章名称”
新建页面 hexo n page “页面名称”
更新博客
hexo clean
hexo g
hexo d
hexo g -d

git常用命令 (👴实在是记不住）

命令 作用
git add . 把所有文件加入缓存区
git commit -m “commit”
git push 推送
